7. CLI 시작 명령어

# 설치
npm install -g @anthropic-ai/claude-code

# 실행
claude # 새 세션 시작
claude -c # 마지막 세션 이어가기
claude --resume <id> # 특정 세션 이어가기
claude "질문 내용" # 한 번만 질문하고 종료
